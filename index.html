about

Finance advice without the jargon

Personal finance is full of intimidating terms — SIPs, mutual funds, emergency funds, expense ratios — and most people avoid planning their money simply because the language feels like it's not meant for them.

MoneyMate AI removes that barrier. A single, purpose-built prompt instructs the model to act as a personal finance advisor, explain concepts with real examples, recommend budgeting and saving strategies, flag investment risk honestly, and never promise guaranteed or unrealistic returns.

model = "gemini-2.5-flash"
temperature = 0.6
framework = "LangChain"
prompt_template = ChatPromptTemplate
ui = "Streamlit"

technical capabilities

What it does, under the hood

Prompt-engineered persona

LANGCHAIN

A single ChatPromptTemplate locks the model into a consistent finance-advisor persona with explicit response rules and formatting instructions.

prompt = ChatPromptTemplate.from_template(template)

Composable LCEL chain

LCEL

Prompt and model are composed with the LangChain pipe operator, keeping the generation logic to a single declarative line.

chain = prompt | model

Gemini 2.5 Flash inference

GOOGLE GENAI

Runs on Google's fast Gemini 2.5 Flash model via langchain_google_genai, tuned with a moderate temperature for advice that's consistent but not robotic.

GoogleGenerativeAI(model="gemini-2.5-flash", temperature=0.6)

Secure config via dotenv

ENV

API credentials are loaded from environment variables at startup, keeping secrets out of source control.

load_dotenv()

Custom Streamlit theming

CSS INJECT

Native Streamlit chrome is hidden and replaced with a hand-styled dark gradient UI, custom inputs, buttons, and chat bubbles via injected CSS.

st.markdown(css, unsafe_allow_html=True)

Guided sidebar & examples

UX

A persistent sidebar lists what the assistant helps with, and an expander surfaces ready-made example questions to lower the barrier to a first query.

st.expander("💡 Example Questions")
responsible by design

Guardrails built into the prompt

The system prompt is explicitly constrained to avoid overpromising on financial outcomes.

⚠️ Educational guidance, not financial advice. MoneyMate AI explains personal finance concepts and encourages healthy money habits, but it never guarantees profits or unrealistic returns, and it flags investment risk clearly whenever it's relevant. For decisions involving real money, it's not a substitute for a licensed financial advisor.
